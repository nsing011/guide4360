                         HOW IT WORKS
================================================================================

1. FIXED WIDTH
   └─ w-40 = 160px (Tailwind class)
   └─ All URL columns same width ✅

2. TRUNCATION
   └─ truncate class adds ellipsis (...)
   └─ Long text replaced with: https://adf.azure.com/...
   └─ Remaining URL hidden but preserved ✅

3. TOOLTIP
   └─ title attribute shows full URL on hover
   └─ User sees complete URL when hovering ✅

4. STILL CLICKABLE
   └─ Link remains functional
   └─ Opens in new tab when clicked ✅

                       TAILWIND CLASSES USED
================================================================================

w-40:           ← Width 40 units (160px)
overflow-hidden ← Hide content beyond width
block:          ← Display as block (for truncate)
truncate:       ← Add ellipsis to long text

white-space: nowrap;
